Commands:
    echo <text>: print a text
    mkdir <dir>: create a directory
    rmdir <dir>: remove a directory
    rm <file>: remove a file
    cd <dir>: change directory
    ls: list files in current directory
    date: print current date
    clear: clear screen
    help: print this help
    exit: exit the shell
    guiinit : initialize the gui
    pide : gui ide
    calc : gui calculator
    file <file> : parse and execute a command file

Args:
    --no-clear | -nc: do not clear the screen when starting BT
    --interpret <file> | -i <file>: interpret a file
